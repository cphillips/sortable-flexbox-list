Pre-ordered List

As a string
<tomalec-sortable-list order="4,3,2,0,1">

Please note that order is a sequence of elements' order. So the key is index from HTML, and value is CSS order - position on screen.
  • zero
  • one
  • two
  • three
  • four
    •
    tomalec-sortable-list.order
    order of sorted items
    tomalec-sortable-list@change
    event triggered after every change
